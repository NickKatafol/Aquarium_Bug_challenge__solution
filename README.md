# Aquarium

### Project Objective

Deep implementation the Vue.js Transition.


### Description

There is the Aquarium.

- If you click on the button "Add the random fish" the Fish is appired moving along a random direction.
- If you click on the Fish it will disapire.


### Stack

Vue.js v3

Vue.js Transition

Typescript

e2e Test (Cypress)



### Setup

```sh
npm install

npm run dev

```

### E2e Test

```sh
npm run test:e2e:dev
```
